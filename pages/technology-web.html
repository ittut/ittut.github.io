  • lighttpd: configure multiple sites on Debian

    Create www folders for each site:

    # mkdir -p /home/lighttpd/site1.com/www
    # mkdir -p /home/lighttpd/site1.com/www

    As long as default lighttpd user and group is www-data change owner and group of www folder to www-data:

    # chown -R www-data:www-data /home/lighttpd/site1.com/www
    # chown -R www-data:www-data /home/lighttpd/site2.com/www
