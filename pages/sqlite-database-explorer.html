SQLite: Database Explorer & sqlite3

7th July 2012    

Few words about SQLite

SQLite is a well-liked preference as database for embedded devices and data storage on client side in application software. SQLite is extremely widely used RDBMS, because it is used today by numerous well-known operating systems, browsers and embedded devices.

SQLite3 Utility

A separate program called sqlite3 is provided that can be used to manage a database, set tables' parameter in database, view, insert and update rows, execute SQL queries and manage a SQLite standalone database file (all operations that are supported by SQLitePlus Database Explorer). This program is a one executable file on the computer. It also is good example of developing programs that utilize the SQLite library. However it is much easier to use SQLitePlus Database Explorer.

Concise and clear excerpt from man page of sqlite3:

SQLite: SQLitePlus Database Explorer

QLitePlus Database Explorer is another and more advanced toll/utility for SQLite databases management.

QLitePlus Database Explorer can be effectively used to amend data in SQLite (with extension .db3) databases.  SQLitePlus Database Explorer is like SQL Server Query Analyzer.  A developer can view/create/drop/ Tables, Views, and Indexes. Making SQL statements and queries to view or otherwise amend necessary data. Data are being showed in a grid control.

This allows you to copy export data to files and elsewhere.

SQLitePlus Database Explorer general description

Buying SQLitePlus license you get the full-features Database Explorer application.

In general, SQLitePlus Database Explorer is an important utility for interfacing with SQLite databases.

Features are as follows:

  • Opening unrestricted Query windows
  • SQLitePlus Database Explorer allows opening many databases simultaneously
  • Viewing database schema information
  • Executing selected query part or full query (like in MS Query Analyzer). SQLite Database Explorer is priceless for working-out complex and composite queries
  • Selecting any of the different SQLite DLLs to manage your database (see the COM DLL page for more info about the SQLite DLLs)
  • SQLitePlus Database Explorer allows to manage Stored Scripts.

Back to SQLite

SQLite keeps all the definitions, indices, tables, and the data itself in a particular, standalone file on a host computer. This can be achieved by using simple design. Whole database is locked while IO writing procedure, however many SQLite read operations can run simultaneously. One of the reasons why SQLitePlus Database Explorer is interfacing with SQLite database easily. Unlike sqlite3 SQLitePlus Database Explorer has its own GUI.
